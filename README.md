# responsive-education-netease

#### This is the responsive one  
该页面用于介绍网易教育产品，能够获取课程，并展示出来。功能点如下：
- 提供登录界面，对用户密码进行MD5加密。
- 操作Cookie保存用户的登录状态和顶部提示的状态。
- 提供淡入淡出的轮播图效果和弹窗介绍视频。
- 通过Ajax获取线上数据以展示课程，每门课程提供 hover 卡片 效果，并巧用z-index 栈节省了课程图片的再加载。
- 提供两种布局尺寸——宽屏、窄屏，并滚动右侧热门课程以更好的展示。
